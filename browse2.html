  • action_happens_in
  • "It relates an Action to the Sequence, Scene or Shot in which it happens."

  • Range: every value of this property is a Sequence, Scene, Shot.
  • > Inverse of: contains_action
  • has_agent
  • "It relates an Action (e.g. murder, chasing, throwing an object) to the Character who does it."
  • Domain: Action.
  • Range: Character.
  • > Inverse Property of: is_agent
  • has_patient
  • "It relates an Action (e.g. murder, chasing, throwing an object) to the Character who suffers it."
  • Domain: Action.
  • Range: Character.
  • > Inverse Property of: is_patient

  • Domain: having this property implies being a owl:Thing.

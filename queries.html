QuueRies

As stated in the homepage of this website, the main goal of this work is to enable the interrogation of the inside of a video and to provide viewers with new and enhanced searching opportunities.

The queries have been elaborated within the software Protégé, by means of the useful SPARQL Query tab. The used prefixes reflect all the ontologies that have been interlinked to Focal, thus are:

PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX owl: <http://www.w3.org/2002/07/owl#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>
PREFIX focal: <http://www.semanticweb.org/ontologies/2021/focal#>
PREFIX dc: <http://purl.org/dc/elements/1.1/>
PREFIX foaf: <http://xmlns.com/foaf/0.1/>
PREFIX skos: <http://www.w3.org/2004/02/skos/core#>
PREFIX ma: <http://www. w3.org/ns/ma-ont#>
PREFIX dbo: <http://dbpedia.org/ontology#>
PREFIX onyx: <http://www.gsi.dit.upm.es/ontologies/onyx/ns#>

Below are listed some interesting queries that can be asked thanks to the data manually inserted in focal ontology, both in relation to the content of the chosen movie, Memento, and to the real world.

SELECT ?Director ?Other_movies ?Release_date
WHERE {
focal:Memento skos:director ?Director.
?Director focal:directed ?Other_movies.
?Other_movies skos:releaseDate ?Release_date.
}

Result

SELECT ?Music ?Author ?Other_movies
WHERE
{?Music focal:famous ?x
FILTER regex(?x, "true")}
{?Music focal:music_in_movie ?Movie}
{?Music focal:music_in_movie ? Other_movies }
{?Music focal:composed_by ?Author}
}

Result

SELECT ?Sequence
WHERE { ?Sequence focal:fabula ?x}
ORDER BY ?x

Result

SELECT ?Character ?Action
WHERE {
{?Character focal:main_protagonist ?x
FILTER regex(?x, "true")}
{?Action focal:has_agent ?Character}
}
LIMIT 10

Result

SELECT ?Victims
WHERE {
focal:Leonard_Shelby focal:kills ?Victims .
}

Result

SELECT ?Last_words
WHERE {
focal:Teddy focal:killed_in ?Scene .
?Scene focal:contains_line ?Line .
?Line focal:is_said_by focal:Teddy.
?Line focal:content ?Last_words .
}

Result

SELECT ?Last_words
WHERE {
focal:Jimmy focal:killed_in ?Scene .
?Scene focal:contains_line ?Line .
?Line focal:is_said_by focal:Teddy.
?Line focal:content ?Last_words .
}

Result

SELECT ?Character ?Purpose
WHERE {
?Character focal:manipulates ?Leonard_Shelby .
focal:manipulation focal:made_by ?Character.
focal:manipulation focal:purpose ?Purpose.
}

Result

SELECT ?Ill ?Fakers ?Disease
WHERE {
{?Ill focal:disease ?Disease .}
UNION {?Fakers focal:faked_disease ?Disease .}
}

Result

SELECT ?Clothes ?Owner ?Description
WHERE {
?Clothes focal:worn_by focal:Leonard_Shelby.
?Clothes focal:owned_by ?Owner.
?Clothes focal:description ?Description.
}

Result
